# Distribución de los datos

<h3>Variables discretas y continuas.</h3>

-   **Continua:** Cuando puede tomar cualquier valor dentro de un intervalo.
    
-   **Discreta:** Cuando no puede tomar ningún valor entre dos consecutivos.

## Cómo determinar la distribución
1. Realizar una representación gráfica de tus datos.
2. Descarta primero lo que no puede ser.
	1. Si hay algún pico en el conjunto de datos, no puede ser una distribución uniforme discreta
	2. Si los datos tienen más de un pico, no es Poisson o binomial.
	3. Si tiene una sola curva, no hay picos secundarios, y tiene una pequeña pendiente en cada lado, podría ser una distribución Poisson o gamma. Pero no podrá ser una distribución uniforme discreta.

Una forma que me gusta pensar en las variabes es que:

Un número continuo no tiene un número máximo definible.  
Un número discreto tiene un número máximo identificable.
