# Crea un menú a tu reporte

**Notas:**  
**Creación del Menú inicial**  
* Ocultar cierta cantidad de columnas  
* Ocultar cierta cantidad de filas  
* Vista: Mostrar  
* Quitar línea de cuadrícula  
* Quitar encabezado  
* Quitar barra de fórmulas  
* Rellenar el espacio de trabajo de un color  
* Insertar: Ilustraciones: Formas  
* Elegir el estilo de la forma  
**Creación de botón redireccionándolo a una hoja**  
* Clic derecho en cuadro  
* Seleccionar vinculo  
* Lugar de este documento  
* Elegir la hoja  
* Se crea el botón  
* Ocultar hojas para obligar a las personas a utilizar los botones  
**Creación de botón redireccionándolo a un archivo exterior**  
* Clic derecho en cuadro  
* Seleccionar vinculo  
* Archivo o página web existente  
* Carpeta actual  
* Documento  
* Se crea el botón
